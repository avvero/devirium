CLI proxy that reduces LLM token consumption on common dev commands. https://github.com/rtk-ai/rtk

Из агента:
- rtk gain — посмотреть аналитику экономии токенов
- rtk gain --history — историю команд с экономией
- rtk discover — найти упущенные возможности в истории
- rtk proxy <cmd> — выполнить команду без фильтрации (для дебага)
- rtk --version / which rtk — проверить установку
